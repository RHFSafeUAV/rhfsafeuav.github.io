Architecture of automated building inspection using the aerial images captured using UAV. The odometry information of UAV is also used for the quantification of different parameters involved in the inspection.

Overview

  • We automate the inspection of buildings through UAV-based image data collection and a post-processing module to infer and quantify the details which helps in avoiding manual inspection, reducing the time and cost.
  • We introduced a novel method to estimate the distance between adjacent buildings and structures.
  • We developed an architecture that can be used to segment roof tops in case of both orthogonal and non-orthogonal view using a state-of-the-art semantic segmentation model.
  • Taking into consideration the importance of civil inspection of buildings we introduced the software library that helps in estimating the distances between adjacent buildings, estimate plan-shape area and object on teh rooftop, and estimating the roof layout.

Modules

In order to estimate the seismic structural parameters of the buildings the following modules hav been introduced:

  • Distance between Adjacent Buildings
  • Plan Shape and Roof Area Estimation
  • Roof Layout Estimation

Distance between Adjacent Buildings

This module provide us the distance between two adjacent buildings. We sampled the images from the videos captured by UAV and performs the panoptic segmentation using state-of-art deep learning model, eliminating any vegetations (like trees). The masked images are then input to a state-of-the art image based 3D reconstruction which outputs a dense 3D poiunt cloud. We then apply RANSAC algorithm for fitting planes between the segmented structural point cloud. Further the points are sampled on these planes to calculate the distance between the adjacent buildings at different locations.

Plan Shape and Roof Area Estimation

This module provides information regardiung the shape and roof area of the building. We segmented the roof using a state-of-the-art semantic segmentation deep learning model. We subjected the input images to a pre-processing module removes distortions from the wide-angle images. then the Data augmentation technique is used to increase the robustness and performance. Hence, later the roof area is calculated using some equation.

Roof Layout Estimation

This module provides information about the roof layout. Since it is not possible to capture the whole roof in a single frame specially in the case of large sized buildings. Hence, we perform large scale image stitching of partially visible roofs followed by non-structural elements detection and roof segmentation.
